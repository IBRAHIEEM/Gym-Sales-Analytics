#  💪 Gym-Sales-Analytics  


An interactive **Power BI** dashboard developed to analyze the sales performance of a fitness products company. The dashboard provides business insights through interactive visualizations, helping decision-makers monitor performance, identify trends, and support data-driven decisions.

---

# 📌 Project Overview

This project demonstrates a complete Business Intelligence solution using **Power BI**. It transforms raw sales data into interactive dashboards that analyze revenue, customers, products, time trends, returns, and sales by location.

---

# 🎯 Business Objectives

- Monitor overall business performance.
- Analyze revenue and profitability.
- Understand customer purchasing behavior.
- Evaluate product performance.
- Track sales trends over time.
- Analyze returned products.
- Compare sales across different locations.
